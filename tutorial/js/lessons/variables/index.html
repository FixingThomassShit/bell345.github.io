Variables and Data Types

Variables can be thought of as containers for information. Instead of having to write and replace the same number or word to be used throughout your program many times, you can edit it once, place it inside a variable, and use that variable in those places. They can be given almost any name, except for a few words called "keywords" that have special meaning within JavaScript.

To use a variable, you must declare it. Declaring a variable is as simple as writing the word var (a keyword), then your variable's name. For example, to declare a variable called myvar:

var myvar;

To actually use a variable, you must assign a value to it using the = operator. For example, to assign a number to myvar:

var myvar = 24;

You can also assign a variable in the same line that you declare it, as above.

Since variables are containers for information, you can substitute values in your code for them. To add this number that you have given your variable to another, just use the variable name instead of a number.

// Equals 30 24 + 6; // Also equals 30 myvar + 6;

There are a few simple data types you can assign a variable:

  • Boolean: Either true or false. These are used for simple functions, such as isNaN(), which returns whether or not a variable is not a number in a boolean.
  • Integer: A number without a decimal point. For example: 42, 8842 or -27.
  • Floating-point number: A number with a decimal point. For example: 82.3, -22.8843 or 664.00.
  • String: Some text. Is delimited (or starts and ends with) with single or double quotes. For example: 'cheese', "Cracking job, old man!" or "All of the time, I like eating some cheese.".
