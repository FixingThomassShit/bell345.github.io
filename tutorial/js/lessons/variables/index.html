Variables and Data Types

Variables can be thought of as containers for information. Instead of having to write and replace the same number or word to be used throughout your program many times, you can edit it once, place it inside a variable, and use that variable in those places. They can be given almost any name, except for a few words called "keywords" that have special meaning within JavaScript.
