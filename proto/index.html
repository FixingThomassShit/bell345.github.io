tXtEng

Version 0.2.1

After seeing the joy of command line interface games and controls, I decided to create a platform that would make it easier to develop these kind of programs for the Web. At the moment, it is in early alpha, so it's featureless. At the moment (v0.2.1), I have implemented:

  • A suitable command line interface
  • Message Of The Day system
  • Command based control protocol
  • The ability to print to the console (print command)
  • Framework for programs (run command) and a test program (flip)
  • A few other utility commands (cl, help)

It's not that much at the moment, but the basic structure exists.

Space Shooter

Version 1.0

Using the free version of Unity Editor, I followed a basic tutorial to make a top-down space shoot-em-up, to get a feel for Unity and how it works.

About the game: WASD to move, Left-Click to fire, and R to restart once you explode. Simple and fun (to make, at least). I must note that this is at least 20 times better looking, more workable and more fun to play with than my most advanced Game Maker game. Bear in mind - I had the same budget.
